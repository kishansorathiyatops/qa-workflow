---
name: qa-workflow prepare-data
description: Create and organize test data
type: command
---

Create and organize test data for test execution with the following steps:

1. **Data Requirements**
   - Identify test data requirements
   - Determine data types and formats
   - Define data constraints

2. **Data Creation**
   - Generate test data sets
   - Create test user accounts (if needed)
   - Prepare test environments

3. **Data Organization**
   - Organize test data by test case
   - Document data setup steps
   - Create data reset procedures

## Outputs
- Test data specification document
- Test data sets (CSV/JSON format)
- Data setup and cleanup scripts
- Test environment configuration
