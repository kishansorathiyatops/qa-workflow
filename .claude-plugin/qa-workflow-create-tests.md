---
name: qa-workflow create-tests
description: Generate test cases from requirements
type: command
---

Generate comprehensive test cases from requirements with the following steps:

1. **Test Case Creation**
   - Create test case template
   - Generate test cases from scenarios
   - Define test steps and expected results
   - Assign test case IDs

2. **Validation**
   - Validate test case completeness
   - Check coverage against requirements
   - Verify clarity and consistency

3. **Documentation**
   - Document preconditions
   - Document test data requirements
   - Add notes and references

## Outputs
- Test case document (Markdown format)
- Test case spreadsheet (if needed)
- Coverage matrix
- Test case IDs and metadata
