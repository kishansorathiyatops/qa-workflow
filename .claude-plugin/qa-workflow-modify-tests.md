---
name: qa-workflow modify-tests
description: Modify and update existing test cases
type: command
---

Modify and update existing test cases with the following steps:

1. **Review Current Test Cases**
   - Identify test cases to update
   - Review existing steps and expected results
   - Check dependencies

2. **Apply Modifications**
   - Update test steps
   - Modify expected results
   - Add new test cases as needed
   - Remove obsolete test cases

3. **Validation**
   - Verify modified test cases
   - Check coverage impact
   - Validate consistency

## Outputs
- Updated test case document
- Change log of modifications
- Updated coverage metrics
