---
name: qa-workflow report
description: Generate test execution report
type: command
---

Generate comprehensive test execution report with the following steps:

1. **Data Compilation**
   - Collect test execution results
   - Gather metrics and statistics
   - Compile defect information

2. **Report Generation**
   - Create executive summary
   - Document test coverage
   - Include pass/fail breakdown
   - Detail discovered defects

3. **Analysis & Recommendations**
   - Analyze test results
   - Identify trends
   - Provide recommendations

## Outputs
- Test Execution Report (HTML/PDF)
- Test Summary Metrics
- Defect Summary
- Coverage Report
- Executive Summary
